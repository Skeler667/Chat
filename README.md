# Chat App

# Stak technologies used:

- Javascript (ES6+)
- React
- React Bootstrap
- Redux Toolkit
- REST API
- WebSocket
- Webpack
- vercel deploy
- Rollbar
- i18next
- Toastify
- LeoProfanity

# System Requirements
node v16

# Install

clone this repository
```
git clone https://github.com/Skeler667/Chat
```

install ci
```
make install
```

run app localy
```
make start
```
